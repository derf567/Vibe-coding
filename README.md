# LANIZOL - Webová stránka pre hydroizolácie

Profesionálna webová stránka pre spoločnosť LANIZOL špecializujúcu sa na hydroizolácie plochých striech, balkónov, terás a základových dosiek.

## 📋 Obsah projektu

```
lanizol-website/
├── index.html          # Hlavná stránka
├── styles.css          # Štýly a dizajn
├── script.js           # JavaScript funkcionality
├── PROPOSAL.md         # Detailný návrh projektu
├── README.md           # Tento súbor
└── images/             # Priečinok pre obrázky (vytvorte)
    ├── logo.png
    ├── project1.jpg
    ├── project2.jpg
    ├── project3.jpg
    └── project4.jpg
```

## 🚀 Ako spustiť

1. **Lokálne testovanie:**
   - Otvorte súbor `index.html` v prehliadači
   - Alebo použite lokálny server (odporúčané)

2. **Použitie lokálneho servera:**
   ```bash
   # Ak máte Python 3:
   python -m http.server 8000
   
   # Potom otvorte: http://localhost:8000
   ```

## 📸 Príprava obrázkov

1. Vytvorte priečinok `images/` v hlavnom adresári
2. Pridajte nasledujúce súbory:
   - `logo.png` - Logo LANIZOL (odporúčaná veľkosť: 200x200px)
   - `project1.jpg` až `project4.jpg` - Fotografie projektov
   - Optimalizujte obrázky pre web (max 500KB na obrázok)

**Tip:** Môžete použiť fotografie z Instagram profilu @lanizol_hydroizolacie_striech

## ✏️ Ako upraviť obsah

### Zmena textov
Otvorte `index.html` a upravte texty medzi HTML tagmi:
```html
<h1>Váš nový nadpis</h1>
<p>Váš nový text</p>
```

### Zmena farieb
Otvorte `styles.css` a upravte premenné na začiatku:
```css
:root {
    --primary-color: #2B7FD9;  /* Hlavná farba */
    --secondary-color: #1a5ba8; /* Sekundárna farba */
}
```

### Zmena kontaktných údajov
V `index.html` nájdite sekciu `#contact` a upravte:
- Telefónne číslo
- Email
- Instagram link
